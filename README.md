# Usage
- clone this project
```bash
git clone https://github.com/loctvl842/cacao_care
```

- add to root directory file `.env` with the following content:
```bash
ADAFRUIT_IO_USERNAME = <Your usernmae>
ADAFRUIT_IO_KEY = <Your key>
```
